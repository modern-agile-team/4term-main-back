    <h1>📜 Commit Convention </h1>
    <ul>
      Add - 새로운 모듈이나 파일, 폴더가 추가되었을 때
      Feature - 새로운 기능이 추가되었을 때
      Refactor - 기존의 코드가 수정되었을 때
      Incomplete - 기능 구현이 완료되지 않았을 때
      Fix - 오류를 수정했을 때
      Remove - 파일이나 폴더를 삭제했을 때
      Merge - 최종적으로 pr 검토 후 develop에 병합시킬 때
     </ul>
